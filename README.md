# Expansion_estrategica_BIOGENESYS
Busca realizar un análisis estratégico de expansión para Biogenesys en el contexto postpandemia del COVID-19, evaluando oportunidades de crecimiento en Latinoamérica
El presente proyecto tuvo como propósito realizar un análisis estratégico de expansión para Biogenesys en el contexto postpandemia del COVID-19, evaluando oportunidades de crecimiento en Latinoamérica con base en datos sanitarios, demográficos y ambientales. A través del uso de herramientas de análisis de datos en Python (Pandas, Numpy, Seaborn, Matplotlib) y su integración con Power BI, se procesaron y visualizaron indicadores clave como la incidencia y mortalidad por COVID-19,  condiciones climáticas y factores socioeconómicos.
El estudio permitió identificar tendencias temporales y geográficas, analizar correlaciones entre variables y determinar los países con mayor potencial para la instalación de laboratorios o centros logísticos. Entre los hallazgos más relevantes se destacan las ventajas competitivas de Colombia, Perú y México, por los doatos suministrados.
